# قدرات - لعبة البطاقات الاستراتيجية

لعبة تعليمية استراتيجية قائمة على المعرفة، مصممة لطلاب الثانوية العامة في السعودية. اللعبة تجمع بين المتعة和教育، حيث تعتمد على المعرفة والاستراتيجية وليس الحظ.

## 🎮 المميزات

### 🎯 وضع الكوتشينة القدراتية
- اختر 5 بطاقات من شبكة مرئية
- مزايدة على نقاطك المتوقعة
- الإجابة الصحيحة تحقق لك نقاط
- الإجابة الخاطئة تجمد البطاقة (قيمة 0)
- الفائز هو الأعلى نقطة

### 🔥 وضع بوكر القدراتي
- بناء أفضل يد بوكر من 5 بطاقات
- كل بطاقة تحتوي على سؤال قياس
- خطأ ينزل البطاقة درجة واحدة
- تكاليف مختلفة لكل يد
- استراتيجيات متقدمة للفوز

### 🃏 أنواع البطاقات
- **♠ السيف**: اللفظي العميق - دفاع قوي
- **♥ القلب**: الكمي الحسابي - هجوم قوي
- **♦ الماسة**: السياقي الملتوي - تضليلي
- **♣ النادي**: التحليل والإحصاء - ضغط نفسي

### 🏆 نظام التقدم
- 100 مستوى مع إكس بي progresive
- إنجازات متنوعة
- إحصائيات تفصيلية
- منافسات يومية

## 🚀 التقنيات المستخدمة

- **Frontend**: React 18 + TypeScript
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## 📦 التثبيت

```bash
# Clone the repository
git clone <repository-url>
cd qodrat-card-game

# Install dependencies
npm install

# Run development server
npm run dev
```

## 🎯 كيفية اللعب

### الكوتشينة القدراتية
1. **اختيار البطاقات**: اختر 5 بطاقات من الشبكة المرئية بناءً على معرفتك
2. **المزايدة**: اعلن عن نقاطك المتوقعة
3. **الإجابة**: اكشف البطاقات وأجب عن الأسئلة
4. **النقاط**: الإجابة الصحيحة تحقق نقاط، الخاطئة تجمد البطاقة

### بوكر القدراتي
1. **اختيار اليد**: اختر نوع اليد التي تريد بناءها (Royal Flush, Straight, إلخ)
2. **اختيار البطاقات**: اختر 5 بطاقات من الشبكة
3. **الإجابة**: اجيب عن كل بطاقة
4. **كشف القيمة**: قيمة اليد تعتمد على دقة إجاباتك

## 📊 الأسئلة

تتضمن اللعبة أكثر من 1000 سؤال معتمدة من:
- أسئلة القدرات السعودية (قياس)

- كيمياء، فيزياء، أحياء، نحو، إنجليزي
- جميع الأسئلة مصنف حسب الصعوبة والموضوع
